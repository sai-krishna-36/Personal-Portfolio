# Personal Portfolio Website

Developed a responsive personal portfolio website using HTML, CSS, and JavaScript to showcase my profile, technical skills, projects, certifications, and contact information. Implemented a clean and user-friendly interface with responsive design principles to ensure compatibility across desktop and mobile devices. The website serves as a centralized platform to present my work, provide resume access, and demonstrate front-end development fundamentals.

### Technologies Used

* HTML5
* CSS3
* JavaScript

### Key Features

* Responsive and mobile-friendly design
* About Me and Skills sections
* Projects showcase
* Resume download functionality
* Contact information section
* Interactive navigation and UI elements
